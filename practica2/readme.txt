La practica consta de 2 programas disponibles el primero es certificador.py y lo que hace es que le
das el nombre del archivo donde tienes un ejemplar y el nombre de un archivo de salida y te crea un 
certificado para ese ejemplar en el archivo de salida y el segundo es el verificador.py y lo que hace 
es que recibe el nombre del archivo donde tienes un ejemplar y el nombre del archivo donde tienes el
certificado.

Ya para compilar te paras en la carpeta src y para crear un certificado simplemente ejecutas
python3 certificador.py
y despues de hacerlo te pedira que pongas un archivo para facilitar la vida ya te imprimo el nombre
de los archivos para que solo escojas uno de los ejemplares y despues de hacerlo te pedira el nombre 
de un archivo para ahi guardar el certificado

Y para el segundo programa te paras en la carpeta src y para probar el algoritmo verificador ejecutas
pyhton3 verificador.py
Despues de hacerlo te pedira que pongas un archivo al igual que el anterior te imprimo el nombre de
los archivos para que solo escojas uno de los ejemplares y despues de hacerlo te pedira un archivo
donde tengas un certificado igual ya te imprimi el nombre de estos para que solo pongas uno 

Y eso es todo, si quieres un ejemplo puedes ejecutar
pyhton3 verificador.py
